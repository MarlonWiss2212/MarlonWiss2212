I'm **Marlon Wisskirchen**, a **frontend-focused full-stack developer** at [denkwerk](https://denkwerk.com) in Cologne. I build modern, responsive web applications with **Nuxt**, **Vue**, and **TypeScript**, caring most about clean implementation, performance, accessibility, and maintainable design systems. The work doesn't stop at the UI: I also build **NestJS** backends and own the delivery side with **AWS**, **Pulumi**, **Docker**, and **Kubernetes**. Self-taught since I was 12, I've grown from web basics into production full-stack work — and I'm always learning something new.
